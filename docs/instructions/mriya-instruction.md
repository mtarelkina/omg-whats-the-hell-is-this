# Инструкция по добавлению папок в мессенджере Телеграм Web

Это пошаговая инструкция о функциональности Telegram

1. Откройте главное окно веб-интерфейса мессенджера.

2. Для вызова меню настроек нажмите на кнопку ![Кнопка](../images/1.png) в левом верхнем углу главного окна и в появившемся меню выберите пункт «Настройки».

![Меню](../images/4.png)

3. В открывшейся панели «Настройки» выберите пункт «Папки с чатами». 

![Папки с чатами](../images/5.png)

Будет открыта одноименная панель.

4. В панели "Папки с чатами" нажмите кнопку **+ Создать новую папку** для перехода к панели «Новая папка», на которой введите наименование создаваемой папки в поле «Название папки». 

![Новая папка](../images/6.png)

5. Далее, в блоке данных «Выбранные чаты», расположенном ниже, нажмите кнопку **+ Добавить чаты** для открытия панели «Выбранные чаты».

![Выбранные чаты](../images/7.png)

6. В списке чатов установите флаги у наименований чатов, которые хотите добавить в создаваемую папку, после чего нажмите кнопку ![Кнопка](../images/2.png) в правом нижнем углу панели для возврата к панели "Новая папка".

7. Нажмите кнопку ![Кнопка](../images/2.png) в правом верхнем углу панели «Новая папка» для возврата к панели «Папки с чатами» и кнопку ![Кнопка](../images/3.png) для возврата в раздел «Настройки».

8. В разделе «Настройки» так же нажмите кнопку ![Кнопка](../images/3.png) для возврата к главному окну веб-интерфейса.

Добавленная папка появится в списке папок в верхней части списка чатов.
